# CV Landing (Astro + Tailwind + JSON i18n)

Static CV landing page migrated to:

- Astro
- Tailwind CSS
- JSON-based translations (`pl`, `en`)

## Local development

```bash
npm install
npm run dev
```

## Build static pages

```bash
npm run build
```

Build output:

- `dist/index.html` (Polish, default language)
- `dist/en/index.html` (English)
